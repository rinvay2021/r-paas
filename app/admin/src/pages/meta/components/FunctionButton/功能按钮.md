### 功能按钮模块开发

整体请参考 BaseField 目录。有一个新建功能按钮的表单。

表单属性如下

```js
  @IsString()
  @IsNotEmpty({ message: '应用编码不能为空' })
  appCode: string;

  @IsString()
  @IsNotEmpty({ message: '元数据对象编码不能为空' })
  metaObjectCode: string;

  @IsString()
  @IsNotEmpty({ message: '操作按钮名称不能为空' })
  buttonName: string;

  @IsString()
  @IsOptional()
  buttonDesc?: string;

  @IsString()
  @IsNotEmpty({ message: '操作按钮编码不能为空' })
  buttonCode: string;

  /** 排序 */
  @IsNumber()
  @IsNotEmpty({ message: '排序不能为空' })
  buttonOrder: number;

  /** 帮助类型 */
  @IsString()
  buttonHelpType?: string;

  /** 帮助提示 */
  @IsString()
  buttonHelpTip?: string;

  /** 帮助链接 */
  @IsString()
  @IsOptional()
  buttonHelpLink?: string;

  /** 级别 */
  @IsString()
  @IsNotEmpty({ message: '级别不能为空' })
  buttonLevel: string;

  /** 事件类型 */
  @IsString()
  @IsNotEmpty({ message: '事件类型不能为空' })
  buttonEventType: string;

  /** 事件 */
  @IsString()
  @IsNotEmpty({ message: '事件不能为空' })
  buttonEvent: string;

  /** 配置 */
  @IsObject()
  @IsOptional()
  buttonConfig?: Record<string, any>;
```

其中 appCode、metaObjectCode 是从上下文获取。其他均为表单属性，需要用户输入。

- buttonName 文本输入框
- buttonDesc 文本输入框
- buttonCode 文本输入框
- buttonOrder 非必填 数字输入框
- buttonHelpType 单选

```js
export enum HelpType {
  TOOLTIP = 'tooltip',
  LINK = 'link',
}
export const HELP_TYPE_OPTIONS = [
  { label: '帮助提示', value: HelpType.TOOLTIP },
  { label: '链接', value: HelpType.LINK },
];
```

- buttonHelpTip 文本输入框
- buttonHelpLink 文本输入框 （校验链接）
- buttonLevel 下拉单选

```js
export enum ButtonLevel {
    /** 列表 */
    List = 'List',
    /** 视图 */
    View = 'View',
    /** 列表行 */
    ListRow = 'ListRow',
}
export const ButtonLevelOptions = [
  { label: '视图', value: ButtonLevel.View },
{ label: '列表', value: ButtonLevel.List },
  { label: '列表行', value: ButtonLevel.ListRow },
];
```

- buttonEventType 下拉单选

```js
export enum ButtonEventType {
    /** 系统事件 */
    System = 'System',
    /** 自定义事件 */
    Custom = 'Custom',
}
export const ButtonEventOptions = [
  { label: '系统事件', value: ButtonEventType.System },
  { label: '自定义事件', value: ButtonEventType.Custom },
];
```

- buttonEvent 事件 根据事件类型联动，系统事件下拉单选、自定义事件文本输入框。并且 options 需要根据级别来决定。

```js
// 内置事件需要根据级别来决定
/** 列表行 */
export enum ListRowButtonEvent {
    /** 修改 */
    Update = 'Update',
    /** 删除 */
    Delete = 'Delete',
}
export const ListRowButtonEventOptions = [
  { label: '修改', value: ListRowButtonEvent.Update },
  { label: '删除', value: ListRowButtonEvent.Delete },
];
/** 列表 */
export enum ListButtonEvent {
    /** 新建 */
    Create = 'Create',
}
export const ListButtonEventOptions = [
  { label: '新建', value: ListButtonEvent.Create },
];
/** 视图 */
export enum ViewButtonEvent {
    /** 新建 */
    Create = 'Create',
    /** 导出 */
    Export = 'Export',
    /** 导入 */
    Import = 'Import',
    /** 批量删除 */
    BatchDelete = 'BatchDelete',
    /** 批量编辑 */
    BatchUpdate = 'BatchUpdate',
}
export const ViewButtonEventOptions = [
  { label: '新建', value: ViewButtonEvent.Create },
  { label: '导出', value: ViewButtonEvent.Export },
  { label: '导入', value: ViewButtonEvent.Import },
  { label: '批量删除', value: ViewButtonEvent.BatchDelete },
  { label: '批量编辑', value: ViewButtonEvent.BatchUpdate },
];
```

- buttonConfig 配置 文本输入框，需要校验 json。
